## 단축키

| 기능                             | 단축키                  |
| -------------------------------- | :---------------------- |
| TERMINAL                         | Ctrl + `                |
| Settings                         | Ctrl + ,                |  
| 소스 format                      | Ctrl + ;                |  
| 검색(현재파일)                   | Ctrl + F                |
| 주석                             | Ctrl + /                |
| 검색(전체파일)                   | Ctrl + Shift + F        |
| Open Preview(MD)                 | Ctrl + Shift + V        |
| open in default browser          | Alt + B                 |
| open in specified browser        | Shift + Alt + B         |
| 한줄 복사                        | Shift + Alt + ↑ or ↓    |
| 한줄 삭제                        | Ctrl + shift + k        |
| 한줄 이동                        | Alt + ↑ or ↓            |
| 세로선택                         | Shift + Alt + 드래그    |
