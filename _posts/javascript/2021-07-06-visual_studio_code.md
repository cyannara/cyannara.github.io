# 단축키

| 기능                             | 단축키              |
| -------------------------------- | :------------------ |
| TERMINAL                         | Ctrl + `            |
| Settings                         | Ctrl + ,            |  
| Open Preview(MD)                 | Ctrl + Shift + V    | 
| open in default browser          | Alt + B             | 
| open in specified browser        | Shift + Alt + B     | 
| 주석                             | Ctrl + /            |
| 한줄 복사                        | Shift + Alt + ↑ or ↓ |
| 한줄 삭제                        | Ctrl + shift + k     |
| 한줄 이동                        |  Alt + ↑ or ↓        |
