## Jenkins를 이용하여 소스 배포하기
1. 구성 -> 빌드후 조치 탭 선택
2. Tomcat URL을 변경 "http://EC2퍼블릭IP:8080" 
2. Credentials 변경하고 save
3. Build Now 실행
4. Console output에서 빌드결과확인
5. 브라우저에서 확인  
   "http://EC2퍼블릭IP:8080/컨텍스트이름"
